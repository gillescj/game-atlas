# Game Atlas

Video game search and showcase site using React and Sass.

[Live Site Here](https://game-atlas.colingillespie.dev/)

API used: [IGDB API v3](https://api.igdb.com/)

### Installing and Running

Within root:

```
npm install
npm run start
```
